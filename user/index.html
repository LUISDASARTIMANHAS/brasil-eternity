galera meu celular não conseguiu copiar os símbolos do clã, pra quem enfrenta o mesmo problema que eu: baixei o aplicativo UNICODPAD

na lateral direita há um quadrado com os códigos de cada caracter, basta clicar e inserir os seguintes números de pesquisa:

- para 『』U+3000

- para B U+1D20

- para r U+02B0

o nome ETER é com fonte comum.

obs: os códigos em si marcam apenas a coluna onde o caractere fica e não o próprio, ou seja, é necessário observar e encontrar o caracter desejado.
